Linux 명령어

clear - 비우기 (위로 올리면 다 있음) 

ls - list의 약자
     ls -l 을 가장 많이씀 좀 더 잘 보여주는거 
     ls -al은 숨김 파일까지 보기(정확히는 리눅스에서는 파일 앞에 .을 찍으면 숨기는데 .이 붙은 파일까지 보는거)

alias - 확인해보면 명령어 긴 것들이 자동 설정되어 있는 것도 있고 직접 설정할 수도 있음.

touch - file만들기 

mkdir - 폴더만들기

mv testfile 1234 - testfile의 이름을 1234로 바꾸기 

mv 1234 ./test - 1234를 현재 폴더에 있는 test폴더 안에 넣기 

cd - : 방금 있었던 폴더로 돌아가기 (이전 폴더로 돌아가기)

cd : 그냥 cd는 원래 초기 폴더로 돌아가기

cp 1234 567 - 1234라는 파일을 567이라는 파일로 복사 

rm 1234 : 1234라는 파일을 삭제 , 리무브의 약자. 
rm -i 1234 : 지우기 전에 한번 물어봄. y하면 지워짐 
(그냥 rm으로는 폴더 안지워짐)

rm -rf rrr : rrr이라는 폴더 지움

top : 내 컴퓨터에 현재 돌아가는 프로세스와 퍼센트 q를 눌러서 끔.
htop : top보다 좀 더 깔끔하게 나옴

ps : process의 약자, 프로세스가 돌아가고 있는지 안돌아가고 있는지
ps -ef : ps보다 좀 더 자세히

ps aux : 지금 돌아가고 있는 여러가지가 나옴. 
ps는 현재 실행하고 있는 프로그램을 볼 때 봄 보통

df : disk 정보를 다 보여줌 

du : 파일들이 용량을 얼마나 차지하고 있는지 

du -a : 숨김파일까지 다 나오게 디스크 용량 확인 
du는 그닥 잘 안쓰임

find : 현재 있는 폴더에서 들어갈 수 있는 모든 경로를 다 검색해줌 

find | grep test : test라는게 어느 위치에 있는지 다 확인할 수 있음 

man ls : manual의 약자, ls에 관한 자세한 얘기가 나옴 다른 거 쓰면 다른 명령어 쓰면 다른거에 대한 자세한 얘기가 나옴.

which ls : ls가 어디 있는지 
which python3 : python3가 어디있는지 알려줌 

tail : 문서의 끝자락을 보여주는 명령어 

## /etc/profile 와 ~/.bash_profile의 차이 
/etc/profile은 시스템 전역(모든 사용자들)에 대한 환경설정
~/.bash_profile은 sudo 아이디(centos에서 로그인하는 명령어)를 이용해 로그인할 때 적용되는 개인 설정 파일
